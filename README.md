# homebrew-tap

Homebrew tap for my fork of [git-toolbelt](https://github.com/sdthach/git-toolbelt).

```console
$ brew install sdthach/tap/git-toolbelt          # stable (pinned tag)
$ brew install --HEAD sdthach/tap/git-toolbelt   # tip of main
```

The formula ([`git-toolbelt.rb`](git-toolbelt.rb)) is synced from `packaging/git-toolbelt.rb` in the main repo by its `release.yml` workflow on each `v*` tag. See [maintaining-the-fork](https://github.com/sdthach/git-toolbelt/blob/main/docs/maintaining-the-fork.md).
